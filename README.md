# Social Media Sentiment Analysis with BERTweet

Social media sentiment analysis is a crucial task in natural language processing, which is widely
applied in public opinion monitoring, brand management, financial prediction and other fields. Based
on BERTweet, a pre-trained model dedicated to social media, this report constructs a three-class
sentiment analysis system for tweets (positive/neutral/negative). The system incorporates a complete
data preprocessing pipeline, a configurable training framework, and systematic hyperparameter
comparison experiments. By comparing different loss functions, learning rates and batch sizes, this
study comprehensively analyzes the impacts of various hyperparameters on model convergence speed, training stability and generalization ability. Experimental results show that the combination of
BERTweet and Focal Loss achieves the best performance on the test set.
